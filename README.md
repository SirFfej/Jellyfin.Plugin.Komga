*** ALPHA install, use at your own risk. Strongly suggested you backup Jellyfin data prior to use. ***

Plugin intended to leverage the reading experience for comics provided by Komga while allowing Jellyfin to continue to be a consolidated front end.

Allow metadata share with scheduled and triggered tasks.

Relies on the file transformation plugin found at 'https://github.com/IAmParadox27/jellyfin-plugin-file-transformation'
